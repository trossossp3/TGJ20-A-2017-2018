Form Elements

How we can have a User enter information through a Web Page

There are several different examples of form elements. They allow the user to enter data into a form that can then be used later (normally through JavaScript or by code on a server).

Examples of different form elements would be:

  • Text/Date Fields
  • Checkboxes
  • Radio Buttons
  • Select Lists
  • Text Area
